# 🚀 Featured Projects

## 🧠 Psyness
> Mental health app with empathetic voice and conversational AI.

`React` `ElevenLabs API` `Redis` `n8n`

MVP focused on frontend, integrating ElevenLabs real-time voice AI with backend conversational agents through a responsive React interface.

---


## 🚂 RailSense
> Railway analytics platform for Mexico City and Vienna.

`Next.js` `TypeScript` `Docker` `OCR` `FastAPI`

Collaborated on frontend development, focusing on data visualization components and multilingual support, including English-to-Dutch localization for the analytics platform.

---

## 🎵 NoteBeat
> An AI-powered journal that maps emotional patterns in text entries against Spotify music history to uncover psychological insights.

`Next.js` `Python` `FastAPI` `PostgreSQL` `Spotify API` `TypeScript` `TailwindCSS`

Independent project focused on sentiment analysis and emotional pattern tracking from daily journals, leveraging Redis for session memory and caching, and using Python-based backend pipelines to synchronize Spotify listening history and correlate mood fluctuations with audio features.

---

## 🖼️ ImageLab
> Interactive digital image processing laboratory for real-time image analysis and transformation workflows.

`React` `Flask` `Python` `OpenCV` `NumPy` `Recharts` `REST API`

Personal project focused on building a full-stack digital image processing platform that combines a React-based interactive frontend with Python/Flask backend services powered by OpenCV and NumPy. Implemented real-time image transformation pipelines, multi-window canvas visualization, histogram analysis, geometric transformations, frequency-domain operations, and morphology-based filters through a REST API architecture for dynamic client-server image processing workflows.
